# Expiry Checker App

A mini-project developed for college coursework.
This app checks the expiry dates of products using image processing / machine learning
and alerts users when items are close to expiry.

# Tech Stack:
  Python
  YOLO Object Detection (best.pt model)
  Web-based Interface (index.html)

# Features
  Detect product expiry information from images
  Simple local server to run the model

# Purpose
  Built as a demonstration of how AI can help reduce waste by monitoring item expiry.
